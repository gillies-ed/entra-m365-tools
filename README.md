# Entra / M365 Tools

A collection of browser-based tools for analysing Microsoft Entra ID and M365 data.
All tools run entirely in the browser — no server, no install, no data leaves your machine.

## Tools

| Tool | Description |
|------|-------------|
| [Sign-in Log Analyser](signin-analyser/) | Multi-user Entra ID sign-in log viewer with threat signals and charts |
